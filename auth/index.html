模块说明 核心安全能力
服务定位
TIANYIMC Authentication 用于用户登录认证、令牌签发与校验、访问权限判断等核心安全能力，支持业务系统快速接入统一身份体系。
核心能力 能力清单
  • 用户身份验证（一次性验证码、idP登录方式）
  • Token 生成、刷新与有效期管理
  • 基于角色与策略的权限控制
  • 认证事件记录与安全审计支持
